This website!

Azure CSS HTML Javascript

I've always wanted to have a personal website that I could use as a portfolio/resume, but I never wanted to pay Squarespace or Wix for their website builders. This may not be as beautiful as a Squarespace site but it gets the job done, and so far hasn't cost me a dime. The only thing I've paid for is the domain name, which was around $10 a year ago.

Hankhayes.com is formatted with HTML, styled with CSS (Bootstrap), hosted for free on Azure, and uses a bit of Javascript to provide consistent formatting on mobile devices.

Longhorn Recruiting

A web application that facilitates the entire job recruiting process

ASP.NET Core C# CSS HTML Javascript SQL

MIS 333K is a right of passage as an MIS major at UT Austin. This is the project that my team and I created over the course of about 2 months. Longhorn Recruiting is a job board that allows students and recruiters to interact in a variety of ways. Job recruiters can post jobs, schedule interviews, and search for talent. Students can browse companies, search for jobs, apply to those jobs, and schedule interviews.

Longhorn Recruiting is built with Microsoft's ASP.NET framework and uses the MVC architecture. It was created using HTML, CSS, C#, Javascript, SQL, and Azure. My group and I nearly completed all of the project requirements (sounds funny, but is actually quite rare) and finished in 6 place out of about 30 teams.
